Flow control statements¶

Break and continue statements¶

The break and continue statements are used in loops (while or for) to modify behaviour of the loop. break ends the loop, while continue ends the specific iteration and comes back to the next element.

Example use of break statement:

counter = 0

while True: # Create and infinite loop
    
    counter += 1
    
    if counter > 29:
        break # Exit the loop if counter is at 30 or more
        
print(counter)

30

Example use of continue statement:

for i in range(10):

    if i%2 == 1: # Skip odd values
        continue
        
    print(i)

0
2
4
6
8

Pass statement¶

pass statement is used in the code to “do nothing”.

for i in range(10):
    
    if i%2 == 1: # Do nothing for odd numbers
        pass
    else: # Multiply even numbers by 4
        i *= 4
        
    print(i)
